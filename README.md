# 疫情打卡
---------------------
## 使用方法：

- 点击`fork`这个仓库
- 进入fork的仓库 -> `action` -> 打开workflow
- 进入fork的仓库 -> `settings` -> secret -> 设置以下secret（命名为config，替换文字）

```
[
  ["翱翔账号","密码"]
]
```
